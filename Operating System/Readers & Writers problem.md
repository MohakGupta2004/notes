When accessing a critical section if a Reader and a writer try to access it at a same time then 
it creates a issue called the reader writers problem. ^przb1UNs

Solution: ^VG8jGHYO


int rc = 0 // For mainting readers count
Semaphore mutex = 1 //For making sure that every process 
can access the critical section at once (discussed in Semaphore)
    
Semaphore db = 1 // Critical Section (Example database access)    

void Reader() {
  while(true){
    down(mutex)
    rc = rc + 1;
    if(rc==1) then down(db)
    up(mutex)

    
    down(mutex)
    rc--;
    if(rc == 0) then up(db)
    up(mutex)
  }
} ^0wywkCsy
